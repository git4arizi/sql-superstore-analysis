# Sales Analysis Insights

## Project Goal
Analyze sales performance using SQL to identify trends, top-performing categories, and regional profitability.

## Key Insights

1. Technology is the highest revenue-generating category.
2. Certain cities contribute significantly more sales than others.
3. Some regions generate high sales but lower profit margins.
4. A small number of sub-categories generate a large share of revenue.

## Skills Demonstrated

- SQL aggregation (SUM, AVG)
- GROUP BY analysis
- Sorting and ranking with ORDER BY
- Business data interpretation
