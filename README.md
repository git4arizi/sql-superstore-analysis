# SQL Sales Analysis Project

## Project Overview
This project analyzes retail sales data using SQL to uncover business insights such as top-performing categories, regional profitability, and high-revenue cities.

## Tools Used
- SQL
- MySQL
- MySQL Workbench

## Dataset
The project uses the Superstore dataset containing information about orders, customers, products, and sales performance.

## Project Structure

dataset/  
Contains the raw Superstore CSV dataset.

schema/  
SQL script used to create the database table.

sql_queries/  
SQL queries used to analyze sales performance.

insights/  
Business insights derived from the SQL analysis.

## Key Analysis
- Total sales performance
- Sales by category
- Profit by region
- Top cities by revenue
- Best performing product sub-categories

## Skills Demonstrated
- SQL aggregation
- GROUP BY analysis
- Data exploration
- Business insight generation
